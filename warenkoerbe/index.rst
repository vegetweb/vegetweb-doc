Warenörbe sind das zentrale Datenmodell von Vegetweb 2.0. Sie ermöglichen den Datenaustausch zwischen verschieden Parteien.
Zu einen Warenkorb gehören folgende Daten:

  * Eine eindeutige Id, mit der der Warenkorb identifizeirt wird
  * Auswahlkriterien für die Plots
  * Die ausgewählten Plots
  * Freigaben zu geschützten Plots
  
Id
--
Ein Warenkorb erhällt sobald er erstellt worden ist, eine eindeutige Id, die sich nicht mehr ändert. Sie bleibt auch erhallten wenn der 
Warenkorb umbenannt wird. Mit dieser Id kann z.B. ein Link auf den Warenkorb gesetzt werden. Hat der Warenkorb z.B. die Id "d60d4962-4a43-42f6-a1c6-80756887afda"
so lautet der Link: https://www.vegetweb.de/#!/warenkoerbe/d60d4962-4a43-42f6-a1c6-80756887afda
