Warenkörbe sind das zentrale Datenmodell von Vegetweb 2.0. Sie ermöglichen den Datenaustausch zwischen verschiedenen Parteien.
Zu einem Warenkorb gehören folgende Daten:

  * Eine eindeutige ID, mit der der Warenkorb identifiziert wird
  * Auswahlkriterien für die Plots
  * Die ausgewählten Plots
  * Freigabeinformationen für die geschützten Plots
  
ID
--
Ein Warenkorb erhält, sobald er erstellt worden ist, eine eindeutige Id, die sich nicht mehr ändert. Sie bleibt auch dann erhalten wenn der Warenkorb umbenannt wird. Mit dieser Id kann z.B. ein Link auf den Warenkorb gesetzt werden. Hat der Warenkorb die Id "d60d4962-4a43-42f6-a1c6-80756887afda"
so lautet die URI: https://www.vegetweb.de/#!/warenkoerbe/d60d4962-4a43-42f6-a1c6-80756887afda
